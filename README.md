Descrição inicial  

Este projeto consiste no desenvolvimento de um E-commerce (Loja Virtual) com foco em simulação de compras online, utilizando a API pública Fake Store para obtenção de produtos.

O sistema permitirá que usuários naveguem por produtos, adicionem itens ao carrinho, realizem simulações de compra e visualizem informações como preços e opções de pagamento.

O objetivo principal é aplicar conceitos de Quality Assurance (QA), criando cenários de teste para validar funcionalidades essenciais do sistema, como cadastro de usuários, login, manipulação do carrinho e processamento de pagamentos.

A aplicação será desenvolvida em Python, consumindo dados da API https://fakestoreapi.com/products, com foco em simplicidade e organização para facilitar a criação e execução de testes.
